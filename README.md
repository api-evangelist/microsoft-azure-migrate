# Azure Migrate (microsoft-azure-migrate)
Azure Migrate REST API enables programmatic management of migration projects, discovery assessments, and server migrations. It supports discovering on-premises servers, assessing readiness for Azure migration, and executing migrations of VMs, databases, and web apps.
